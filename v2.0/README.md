# SentinelOps v2.0

This directory contains the next-generation SentinelOps architecture. The existing
root application remains the stable v1 release while v2.0 is developed and tested.

## Phase 1: Foundation

- [ ] Modular frontend structure
- [ ] Modular Python backend
- [ ] Versioned `/api/v2` endpoints
- [ ] Central configuration
- [ ] Structured application logging
- [ ] SQLite schema migrations
- [ ] Stronger validation and error responses
- [ ] Automated Python and JavaScript tests

## Planned v2.0 capabilities

Later phases will add Sigma rules, Sysmon-focused analysis, incident investigations,
allowlists, false-positive management, and expanded correlation.

## Status

Phase 1 is in development. Use the stable application in the repository root until a
v2.0 preview release is announced.
